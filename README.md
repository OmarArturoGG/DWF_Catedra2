# Biblioteca Digital - Sistema de Préstamo de Libros

## Requisitos previos
- Java 17
- MySQL
- Maven

## Instalación

1. Clonar el repositorio
2. Cambiar en el archivo application.properties la contraseña de su MySQL en la linea 3
